  (`NexaFileGrid.xaml.cs` `OnResizeStart/Move/End`). 핸들 폭 8px.
- **헤더 색상 구분**: 헤더 Border `Background="#18FFFFFF"` + `BorderBrush="#33FFFFFF"`(연한 배경/경계).
- **정렬 상태 컬럼서 분리**: `NexaGridColumn`은 너비/키/정렬가능 여부만 보유.
  정렬 상태는 **패널(NexaFileGrid)별**로 관리(좌/우 독립) — 너비만 공유, 정렬은 독립.

## 3. 설계 (docs/23 컬럼 시스템)
- **§3-1 너비 vs 정렬 분리(확정)**: 너비=좌/우 실시간 동기, 정렬=패널별 독립, 폴더별 정렬 상태 저장·복원.
- **§4 정렬(패널별 독립)**: 헤더 클릭=정렬키 토글(▲/▼), **다중 컬럼 정렬**(Shift/Ctrl+클릭, 순번),
  정렬 초기화(열거 순서 복원), `SortRequested` 이벤트로 도메인 비종속 유지.
