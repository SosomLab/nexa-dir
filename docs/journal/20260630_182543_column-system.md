## 2. 정리 (결정 — [23](23-column-system.md))
- **컬럼 = 데이터**: `IColumn`(정의) + `ICellValueProvider`(값) + `ColumnLayout`(순서/너비/표시, 직렬화). 컨트롤은 컬럼 의미 모름(도메인 비종속).
- **종류**: 기본열(VFS 메타, 이름/**확장자 분리 옵션**/크기/날짜/속성/**클라우드 상태**/경로) · **사용자 정의 수식열**(안전 샌드박스 표현식) · 플러그인열.
  - 수식 예: `=size/1024 & " KB"`, `=if(kind="dir","폴더","파일")`, `={modified:yyyy-MM}`. 순수 계산만, 화이트리스트 함수. 코어 평가 우선(가상화로 보이는 셀만).
- **듀얼 동기화 정책(선택)**: 독립 / 실시간 / 수동 적용(좌→우·우→좌·양쪽맞춤). 동기 단위=`ColumnLayout`, `PaneHost`가 정책 적용. **레이아웃 프로파일**(폴더 종류별).
- 정렬은 셀 **정렬키**(표시 텍스트와 별개)로.

## 3. 컴포넌트 구조 요구
- 컬럼 런타임 추가/제거/재정렬/리사이즈/표시토글 + 직렬화.
- 셀 값 위임(기본/수식/플러그인 동일 취급), 보이는 셀만 평가, 비동기 셀 자리표시자→갱신.
