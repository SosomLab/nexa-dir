# 작업 기록 · 탭 닫기(Ctrl+W · 더블클릭) + 더블클릭 동작 설정 (F22)

> 요구: Ctrl+W 탭 닫기 + "탭 더블클릭 시 동작" 설정(없음/닫기/즐겨찾기/팝업, 기본 닫기), 지금은 닫기 구현.

## 구현
- Settings.cs: TabDoubleClickAction(None/Close/Favorite/PopupMenu) + TabOptions.DoubleClick=Close + AppSettings.Tab.
- MainWindow: CloseTab(최소1개 유지·이웃 전환), OnTabDoubleTapped(설정 분기, 기본 닫기; 즐겨찾기/팝업 후속),
  OnGridKeyDown Ctrl+W → 활성 탭 닫기. 탭 위 더블클릭 소비(빈 영역 더블클릭=추가 F20 유지).
- XAML: 탭 아이템 DoubleTapped=OnTabDoubleTapped(좌/우).
- docs/19 F22, 05 FR-B6a, 26 §5-4(Ctrl+W).
