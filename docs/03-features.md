# 03 · 기능 명세

마일스톤별 상세 기능 명세. (요약: [02-roadmap.md](02-roadmap.md))

## 1. 탭 (패널별, M1)

> Double Commander / Path Finder 식 — **각 패널 상단에 자체 탭 스트립**.

| 항목 | 명세 |
| --- | --- |
| 생성/닫기 | Ctrl+T(새 탭), Ctrl+W(닫기), Ctrl+Shift+T(복원) |
| 전환 | Ctrl+Tab / Ctrl+1..9, 중간클릭으로 폴더를 새 탭 |
| 순서 | 드래그로 재배치, 패널/창 간 탭 이동(P1) |
| 라인 모드 | **싱글 라인 ↔ 멀티 라인** 전환 — 탭이 많으면 줄바꿈으로 전부 표시 |
| 히스토리 | 탭마다 독립 뒤로/앞으로 스택 |
| 세션 | 종료 시 탭·경로·스크롤·펼침/선택 저장, 재시작 복원 |

## 2. 듀얼 패널 (M1)

> Double Commander / Path Finder 식 — **좌/우 분리 패널이 각각 다른 디렉터리** 탐색,
> 각 패널은 자체 탭 스트립(§1)을 가짐.

| 항목 | 명세 |
| --- | --- |
| 레이아웃 | 좌우/상하 분할, 스플리터로 비율 조정, 단일↔듀얼 토글 |
| 포커스 | 활성 패널 강조, Tab으로 패널 이동, **패널마다 독립 탭 스트립** |
| 전송 | F5 복사 / F6 이동(활성→반대 패널), 드래그앤드롭 |
| 동기 | 같은 폴더 열기, 양쪽 동기 스크롤(옵션) |

## 2-X. 인라인 폴더 확장 + 폴더 교차 다중 선택 (M1) ★ 플래그십

| 항목 | 명세 |
| --- | --- |
| 인라인 펼침 | 목록/디테일 보기에서 폴더 행 삼각형(▶/▼)으로 현재 위치 유지한 채 펼침/접힘 |
| 진입 vs 펼침 | 더블클릭/Enter=진입(네비게이트), 삼각형·`→`/`←`=인라인 펼침 (공존) |
| 교차 선택 | Ctrl+클릭(비연속), Shift(가시 범위)로 **서로 다른 폴더의 항목 동시 선택** |
| 혼합 작업 | 혼합 선택을 한 번에 복사/이동/삭제/드래그/압축 (작업 엔진 혼합 입력) |
| 상태 보존 | 펼침/선택을 정렬·외부변경(watcher)·세션 복원에도 유지 |
| 성능 | 가시 노드 평면 스트림 + 가상화, 10만 노드 <150ms·60fps |

> 상세 설계·데이터 모델·수용 기준: [07-flagship-tree-multiselect.md](07-flagship-tree-multiselect.md)

## 3. 파일 작업 (M1)

- **작업 종류**: 복사·이동·삭제(휴지통/영구)·이름변경·새 폴더·바로가기.
- **진행률**: 큰 작업은 백그라운드 잡 + 진행률/속도/남은시간, 일시정지·취소.
- **충돌 정책**: 덮어쓰기 / 건너뛰기 / 둘 다 보관(자동 이름) / 더 최신만.
- **안전장치**: 모든 파괴적 작업 Undo, 휴지통 경유 기본, 대량 삭제 확인.
- **일괄 이름변경**: 패턴/정규식/넘버링 + 실시간 미리보기.

## 3-J. 컨텍스트 메뉴 — 셸 통합 + 고유 메뉴 (M1)

> **기본 Windows 컨텍스트 메뉴를 그대로 쓰고, 그 위에 Nexa 고유 항목을 더한다.**

| 항목 | 명세 |
| --- | --- |
| 셸 메뉴 호스팅 | `IShellFolder::GetUIObjectOf` → `IContextMenu`(클래식) / `IExplorerCommand`(Win11 모던) 호출. 서드파티 셸 확장(압축·백신·열기 앱 등) 포함 |
| 고유 항목 병합 | "새 탭/패널에서 열기", "선택 바구니에 추가", "경로 복사", AI 작업 등을 셸 메뉴 앞/뒤·서브메뉴로 병합 |
| 다중·교차폴더 | 혼합 선택(서로 다른 폴더 항목)에서도 메뉴·동작 일관 |
| 액션 공유 | 컨텍스트 메뉴·단축키·명령 팔레트가 **단일 액션 레지스트리** 공유 |

**구현 메모(코어/인터롭):** 셸 메뉴 호출은 Windows 전용 COM 상호운용 → `interop`/UI 계층에서
HMENU 또는 IExplorerCommand 열거를 수집해 우리 메뉴 모델과 합성. 비Windows에서는 비활성(이식 격리).

## 3-K. 퀵 런처 바 (M1) — 통합 터미널 + 외부 프로그램

> 타이틀바 바로 아래 최상단의 **아이콘 런처 영역**. 활성 패널의 현재 경로를 컨텍스트로 사용.

```
┌───────────────────────────────────────────────── 타이틀바 ──┐
├─[퀵 런처]  [▣ cmd] [▣ pwsh] [▣ WT] | [▣ VSCode] [▣ Subl] [＋]─┤  ← 본 기능
├─[주소줄 / 명령바 / 검색]────────────────────────────────────┤
│  좌 패널(탭 스트립 + 트리)        │  우 패널(탭 스트립 + 트리)  │
└─────────────────────────────────────────────────────────────┘
```

| 항목 | 명세 |
| --- | --- |
| 통합 터미널 | cmd / PowerShell / pwsh / Windows Terminal을 **현재 폴더에서** 실행. (P2) 임베디드 터미널 패널 |
| 외부 프로그램 | 에디터 등 등록(경로·인자 토큰 `%path% %selection%`·아이콘) → 폴더/선택 파일로 실행 |
| 등록/감지 | 항목 추가/삭제/순서변경, 아이콘 지정, 설치 앱 자동 감지(선택) |
| 인터랙션 | 클릭 실행, **파일 드래그앤드롭으로 그 프로그램에 열기**, 단축키 바인딩 |
| 액션 공유 | 컨텍스트 메뉴(§3-J)·명령 팔레트와 동일 액션 레지스트리 |
| 프로파일(P2) | 용도별(개발/미디어) 런처 세트 전환 |

**구현 메모:** 실행은 `ShellExecute`/`CreateProcess`로 작업 디렉터리=활성 경로 지정.
인자 토큰 치환(`%path%`, `%selection%`, `%dir%`)으로 선택(교차폴더 포함) 전달. 비Windows는 격리.

## 4. 빠른 미리보기 (M2)

| 분류 | 지원 |
| --- | --- |
| 이미지 | JPEG/PNG/GIF/WebP/HEIC/SVG, EXIF 표시 |
| 문서 | PDF(첫 페이지~), TXT/MD, Office는 셸 썸네일 |
| 코드 | 구문 강조, 큰 파일은 헤드만 |
| 미디어 | 비디오 썸네일/스크럽, 오디오 메타 |
| 동작 | 스페이스바 퀵룩 오버레이 + 사이드 PreviewPane 양쪽 지원 |

- **성능**: 디코딩은 코어 스레드풀, 결과 캐시(메모리 LRU + 디스크). 목표 <100ms.

## 5. 검색 (M3)

- **인스턴트 파일명 검색**: 전역 인덱스(tantivy), 입력 즉시 결과(<50ms).
- **필터 문법**: `type:pdf size:>10mb date:thisweek tag:invoice path:projects`.
- **내용 검색**: 텍스트/코드/문서 전문 인덱싱.
- **결과 뷰**: 결과를 가상 폴더처럼 정렬·작업·미리보기.

## 6. 클라우드 / 원격 (M4)

- **VFS Provider 모델**: `list / stat / read / write / delete / watch` 통일 인터페이스.
- **지원(우선순위)**: SFTP → S3 호환 → OneDrive/Google Drive.
- **자격증명**: Windows Credential Manager / DPAPI 암호화 저장.
- **UX**: 사이드바에 마운트로 노출, 로컬과 동일하게 탭/듀얼/검색 사용.
- **전송**: 원격↔로컬, 원격↔원격(듀얼 패널), 재개 가능·체크섬 검증.

## 7. AI 검색 / 정리 (M5)

- **자연어 검색**: "지난달 회의록", "흐릿한 사진" 등 → 의미 기반 매칭.
- **임베딩**: 파일명/내용/이미지 임베딩, 로컬 ANN 인덱스.
- **자동 정리**: 분류·태깅·중복/유사 탐지, 규칙 추천 후 일괄 적용 미리보기.
- **프라이버시**: 로컬 우선, 외부 API 사용 시 명시적 동의·범위 제한.

## 8. 횡단 관심사 (Cross-cutting)

- **테마**: Fluent/Mica, 다크/라이트, 강조색, 밀도(컴팩트/표준).
- **접근성**: 키보드 완전 조작, 스크린리더, 고DPI.
- **국제화**: 한국어/영어 우선, 리소스 분리.
- **설정**: 단축키 재정의, 기본 뷰/정렬, provider 자격증명.
- **로깅/원격측정**: 옵트인, 크래시 리포트, 성능 트레이스.
