## B. 실행 · 탭 · 포커스

### B1. 파일 실행(ShellExecute) — BUG-003 ★

- 기대: **오류 없이** 각 확장자의 기본 프로그램으로 열림. (이전엔 desktop.ini에서 "unauthorized" 오류)
- [ ] desktop.ini 실행 OK  [ ] 일반 파일 실행 OK  [ ] 폴더 더블클릭=진입(변화 없음)

### B2. Alt+↓ 실행
- 절차: 파일에 캐럿 두고 **Alt+↓**. 폴더에 캐럿 두고 Alt+↓.
- 기대: 파일=더블클릭과 동일 실행 / 폴더=진입.
- [ ] 파일 Alt+↓ 실행  [ ] 폴더 Alt+↓ 진입

### B3. 새 탭 이름 — BUG-005
- 절차: 탭 바 **빈 영역 더블클릭**(새 탭 추가). 새 탭의 **이름 라벨** 즉시 확인.
- 기대: 새 탭 이름이 **처음부터 폴더명으로 표시**(공백 아님). (이전엔 이동 전까지 공백)
- [ ] 새 탭 이름 즉시 표시

### B4. 비활성 패널 활성화 — BUG-004
- 절차: 좌 패널 활성 상태에서 **우 패널의 (1)행 클릭 / (2)빈 공간 클릭 / (3)우클릭** 각각.
- 기대: 세 경우 모두 **우 패널이 활성화**(선택색이 파랑, 좌 패널은 회색). 키보드 방향키가 우 패널에 작동.
- [ ] 행 클릭 활성화  [ ] 빈 공간 클릭 활성화  [ ] 우클릭 활성화

## C. 선택 · 이름변경 · 마우스 (B-8/B-9m) — 회귀 주의 영역

### C1. 단일/다중/범위 선택(회귀 확인)
- 절차: (1)항목 클릭=단일 / (2)Ctrl+클릭 다른 항목=추가(다중) / (3)Shift+클릭=범위.
- 기대: 각각 정상. Ctrl+클릭으로 **여러 개 파란 하이라이트 유지**.
- [ ] 단일  [ ] Ctrl 다중  [ ] Shift 범위

### C2. 이름변경 트리거 — 드래그/우클릭 오발동 없음 — BUG-002
- 절차: (1)선택된 항목을 **드래그 시도**(누른 채 이동) / (2)항목 **우클릭**.
- 기대: **둘 다 이름변경이 시작되지 않음**. 드래그=드래그 시작, 우클릭=컨텍스트 메뉴.
- [ ] 드래그 시 이름변경 안 뜸  [ ] 우클릭 시 이름변경 안 뜸

### C3. 이름변경 정상 동작
- 절차: 항목 클릭(선택) → 잠깐 후(≈0.5초+) **같은 항목 재클릭** / 또는 **F2**.
- 기대: 인라인 편집기(테두리만, 배경 투명, 원래 폰트색) 표시. Enter=확정·ESC=취소·영역밖 클릭=확정. **빠른 더블클릭은 진입/실행**(이름변경 아님).
- [ ] 재클릭 편집 시작  [ ] F2 편집  [ ] 빠른 더블클릭=진입(편집 아님)  [ ] Enter/ESC/영역밖

### C4. 다중 선택 드래그 — B-9m ★
- 절차: **3개 선택**(Ctrl+클릭) → 그 중 **하나를 잡고 다른 폴더로 드래그→드롭**.
- 기대: **3개 모두** 이동. 드래그 시작 시 선택이 1개로 줄지 않음.
- [ ] 3개 선택 드래그 → 3개 이동
- 절차2: 3개 선택 상태에서 그 중 하나를 **드래그 없이 그냥 클릭**(떼기).
- 기대2: 그 항목만 **단일 선택으로 축소**(Explorer식).
- [ ] 다중→단일 축소

## D. 파일 작업 · 단축키 · 컨텍스트 메뉴

### D1. 클립보드 복사/이동
- 절차: 파일 선택 → **Ctrl+C** → 다른 폴더 → **Ctrl+V**. 이어서 **Ctrl+X** → 다른 폴더 → **Ctrl+V**.
- 기대: C+V=원본 유지+사본 생성 / X+V=원본 이동. 이름 충돌 시 " (2)" 부여.
- [ ] 복사(Ctrl+C/V)  [ ] 이동(Ctrl+X/V)  [ ] 이름 충돌 " (2)"

### D2. 삭제 — 휴지통 vs 완전 — B-11r ★
- 절차: 파일 선택 → **Del** / 다른 파일 → **Shift+Del**.
- 기대: Del=**휴지통으로**(확인창 없음, 휴지통에서 복구 가능) / Shift+Del=**확인창** 후 완전 삭제.
- [ ] Del=휴지통(복구 가능)  [ ] Shift+Del=확인창→완전삭제

### D3. 컨텍스트 메뉴 분리 — B-10c ★
- 절차: (1)**폴더** 우클릭 / (2)**파일** 우클릭 / (3)**빈 영역** 우클릭.
- 기대:
  - 폴더: 열기 · 잘라내기 · 복사 · **폴더에 붙여넣기** · 삭제(휴지통) · 완전삭제 · 이름 바꾸기
  - 파일: 실행 · 잘라내기 · 복사 · (붙여넣기 **없음**) · 삭제(휴지통) · 완전삭제 · 이름 바꾸기
  - 빈 영역: 붙여넣기 · 새로고침(F5)
- [ ] 폴더 메뉴(폴더에 붙여넣기 有)  [ ] 파일 메뉴(붙여넣기 無)  [ ] 빈영역 메뉴
- 절차2: 클립보드에 복사해둔 상태에서 **폴더 우클릭 → "폴더에 붙여넣기"**.
- 기대2: 그 **폴더 안으로** 붙여넣기(현재 폴더 아님).
- [ ] 폴더에 붙여넣기 = 그 폴더로

### D4. 새로고침
- 절차: **F5** / 빈 영역 우클릭 → 새로고침.
- 기대: 현재 폴더 재열거(펼침 상태 유지).
- [ ] F5  [ ] 메뉴 새로고침

## E. 드래그 앤 드롭

### E1. 폴더 이동 + 자동 스크롤 — B-11d
- 절차: 파일을 **다른 폴더 행 위로** 드래그→드롭. 항목 많은 폴더에서 드래그한 채 **목록 위/아래 가장자리**에 머무름.
- 기대: 그 폴더로 이동. 가장자리에서 **자동 스크롤**(계속 머물면 계속 스크롤).
- [ ] 폴더로 이동  [ ] 위 가장자리 자동 스크롤  [ ] 아래 가장자리 자동 스크롤

### E2. 좌우 패널 간 이동 — B-12x
- 절차: 좌 패널 파일을 **우 패널의 (1)폴더 행 / (2)빈 영역**으로 드래그→드롭. 반대 방향도.
- 기대: (1)그 폴더로 / (2)우 패널 현재 폴더로 이동. 양쪽 목록 갱신.
- [ ] 좌→우 폴더행  [ ] 좌→우 빈영역(현재 폴더)  [ ] 우→좌

### E3. 디스크별 기본 + Alt 반전 — B-14dnd ★
- 절차: (1)**같은 디스크** 폴더로 드래그 / (2)**다른 디스크(USB 등)** 로 드래그 — 각각 드롭 전 **캡션** 확인, 그리고 **Alt 누른 채** 확인.
- 기대:
  - 같은 디스크: 기본 **이동**(캡션 "…이동") · Alt 누르면 **복사**
  - 다른 디스크: 기본 **복사**(캡션 "…복사") · Alt 누르면 **이동**
- [ ] 같은 디스크=이동  [ ] 다른 디스크=복사  [ ] Alt 반전(양쪽)  [ ] 실제 결과가 캡션과 일치

### E4. 탭 hover 전환 — B-13t (양호 확인됨)
- 절차: 드래그한 채 **다른 탭 위 2초** 머무름 → 그 탭으로 전환되면 계속 드래그해 드롭.
- 기대: 2초 후 탭 전환, 그 폴더에 드롭 가능. (설정 `TabDwellMs` 기본 2000)
- [ ] 탭 2초 hover 전환  [ ] 전환 후 드롭

### E5. 폴더 hover 진입(spring-load) — B-15h ★

- 기대: **그 폴더로 진입**(목록이 그 폴더 내용으로 바뀜) → 계속 드래그해 하위 폴더에 드롭 가능. (설정 `FolderDwellMs` 기본 3000)
- [ ] 폴더 3초 hover 진입  [ ] 진입 후 하위 드롭

### E6. ESC 취소 — B-14e
- 절차: 드래그 도중 **ESC**.
- 기대: 드래그 취소(아무 이동/복사 없음), 상태바 "드래그 취소". hover 타이머·자동 스크롤 즉시 정지.
