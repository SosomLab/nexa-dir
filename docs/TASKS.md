## 4. 착수 권장 순서(이 배치)

B-8 + B-9m(마우스 재정비, 세트) → B-11r(휴지통 배선) → B-10c(컨텍스트 분리) → B-14dnd·B-15h(DnD 정책·hover; 설계 후 구현) → COL-1(확장자) → COL-2·COL-3(정렬) → B-12w(watcher; 설계 후) → B-13u(undo; nexa-ops와) → SHELL(셸 통합; 별도 ADR).
타입어헤드(TA)는 [docs/32](32-typeahead-find.md) 설계 확정 상태 — 사용자 "구현 시작" 지시 대기.
